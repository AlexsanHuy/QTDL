# Ktx_project  
How to run this project  
1: Install nodejs  
2: run 'npm install'   
3: run 'npm install --save-dev dotenv nodemon' 
4: add .env file with infomation about your database  
  ![image](https://github.com/user-attachments/assets/b14b76a7-2e2c-4e9b-a2ff-6c4ff255da01)

5: run npm start
